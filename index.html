Esports

Prime Bot includes powerful features for managing tags and organizing content. Below are the available commands for TagCheck and EasyTag systems:

TagCheck Commands

  • Add a Channel to TagCheck: ?tagcheck channel add <#channel>
  • Remove a TagCheck Channel: ?tagcheck channel remove <#channel>
  • Show all TagCheck Channels: ?tagcheck channel show
  • Add User to Bypass: ?tagcheck bypass add <@user>
  • Remove User from Bypass: ?tagcheck bypass remove <@user>
  • Show All Bypass Users: ?tagcheck bypass show

EasyTag Commands

  • Add a Channel to EasyTag: ?easytag channel add <#channel>
  • Remove an EasyTag Channel: ?easytag channel remove <#channel>
  • Show all EasyTag Channels: ?easytag channel show
  • Add User to Bypass: ?easytag bypass add <@user>
  • Remove User from Bypass: ?easytag bypass remove <@user>
  • Show All Bypass Users: ?easytag bypass show

Greeting System

Prime Bot's Greeting System allows you to welcome new members to your server with personalized messages.

  • Add a Greeting Channel: ?greet add <#channel>
  • Remove a Greeting Channel: ?greet remove <#channel>
  • List all Greeting Channels: ?greet show
  • Toggle AutoDelete: ?greet autodel <#channel>
  • Test Greeting: ?greet test <#channel>

Media Management

Organize media sharing with Prime Bot's Media Management system.

  • Add a Media Channel: ?media channel add <#channel>
  • Remove a Media Channel: ?media channel remove <#channel>
  • Show all Media Channels: ?media channel show
  • Add User to Bypass: ?media bypass add <@user>
  • Remove User from Bypass: ?media bypass remove <@user>
  • Reset Bypass Users: ?media bypass reset
  • Show All Bypass Users: ?media bypass show

Screenshot Verification

Verify user-submitted screenshots with this feature.

  • Add a Verification Channel: ?ssverify channel add <#channel>
  • Remove a Verification Channel: ?ssverify channel remove <#channel>
  • Show all Verification Channels: ?ssverify channel show

Ticket System

Manage support requests efficiently with Prime Bot's Ticket System.

  • Create a Ticket Panel: ?ticket panel <panel_name>
  • List all Ticket Panels: ?ticket list
  • Delete a Ticket Panel: ?ticket paneldelete <panel_name>
  • Reset Ticket Panels: ?ticket panelreset
  • Add User to Ticket: ?ticket add <user>
  • Remove User from Ticket: ?ticket remove <user>
  • Reopen Ticket: ?ticket reopen
  • Delete Ticket: ?ticket delete
